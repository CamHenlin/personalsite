henlin.org

This is just a random assortment of my projects that I am most proud of or interested in at this time.

iMessageClient

iMessageClient is a nodejs app that allows you to interact with Messages.app via a terminal interface.

iMessageWebClient

iMessageWebClient is a nodejs app that allows you to interact with Messages.app via a web browser.

Executive Man

ExecutiveMan is a nearly 100% feature-complete JavaScript implementation of my favorite NES game. Check it out on the ExecutiveMan GitHub page as well!

pocketga.me

pocketga.me is an online emulator system aimed at allowing users to play NES, GameBoy, SNES, Master System, and more, games on iOS devices without the need to jailbreak.

Touch Polyfill

Touch Polyfill allows developers to use standard JavaScript touch events on Internet Explorer 10 and 11, which only support pointer events.
